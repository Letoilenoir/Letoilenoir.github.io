Quick breakdown of the SKU reference:<br>
    
    string sku = "01-MN-L";
    string[] product = sku.Split('-');

    string type = "";
    string color = "";
    string size = "";

    if (product[0] == "01")
    {
        type = "Sweat shirt";
    } else if (product[0] == "02")
    {
        type = "T-Shirt";
    } else if (product[0] == "03")
    {
        type = "Sweat pants";
    }
    else
    {
        type = "Other";
    }
    

Due to zero indexing the elements in the product serial number are as follows:
* product[0] = "01" (type)
* product[1] = "MN" (color)
* product[2] = "L"  (size)

To convert the <b>"if, else if, else"</b> statement to a <b>"switch"</b> statement refactor the code as follows:

    switch (product[0])
    {
        case "01":
        type = "Sweat shirt";
        break;

        case "02":
        type = "T-Shirt";
        break;
	
        case "03":
        type = "Sweat pants";
        break;
	
        default:
        type = "other";
        break;
    }
